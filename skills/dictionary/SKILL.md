# Dictionary Lookup

Looks up definitions of words and technical terms.

## Usage

Query: "define ephemeral"
Result: "lasting for a very short time"

## Available Words

The built-in dictionary includes common English words and AI/ML technical terms
for reproducible benchmarking: ephemeral, ubiquitous, algorithm, paradigm,
optimize, latency, inference, hallucination, benchmark, perplexity,
tokenization, quantization, transformer, gradient, entropy.
